Problem Statement :                                                                                                                                                In today's digital workspace, 'shoulder surfing' and visual privacy breaches are major security threats. Users often handle sensitive information on apps like WhatsApp, Telegram, or private documents in public or shared spaces. Currently, there is no automated way to protect the screen if a stranger looks over the user's shoulder or if the user leaves their desk while a private app is open.
                                                                                                                                                              Solution Explanation
We developed GhostGaze AI, an intelligent privacy shield that uses Computer Vision and Biometric Profiling.
Key Features:
3-Point Biometric Enrollment:                                                                                                                                     The system captures the owner's front, left, and right facial profiles to ensure recognition from multiple angles.
App-Aware Monitoring:                                                                                                                                             The camera only activates when a designated 'Privacy App' (like WhatsApp or Notepad) is in the foreground, saving power and ensuring camera privacy.
Stranger Detection & Immediate Action:                                                                                                                               If the system detects more than one face or an unauthorized person looking at the screen, it instantly triggers a full-screen Red Privacy Overlay and minimizes all active windows to protect the data.
Silent Operation:                                                                                                                                                   It runs seamlessly in the background without disturbing the user's workflow unless a threat is detected.
